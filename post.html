One-time setup

To publish posts, this page needs permission to update the site. You'll create a free "token" on GitHub — it's like a key that only works for this blog.

  1. Make sure you're signed in to a GitHub account that has access to this blog's repository.
  2. Open github.com/settings/personal-access-tokens/new
  3. Token name: coffee-blog. Expiration: pick 1 year.
  4. Repository access: choose Only select repositories → select numenuke.github.io.
  5. Permissions → Repository permissions → Contents → set to Read and write.
  6. Click Generate token and copy it (starts with github_pat_).

Your token is saved only in this browser, on this device. Each person posts from their own device with their own token.
